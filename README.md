# MEAN Stack NTI Tasks

This workspace is organized by task folders. Each folder contains the solution for the task given in that session, along with any notes or supporting files for that task.

## Folder Convention

- `DayXX-Task/` contains the work for that task day.
- `solve.js` is the main solution file for the task.
- `task-session-*.md` stores the task instructions or session notes.

## How to Run

1. Open the task folder you want to run.
2. Run the solution with Node.js:

```bash
node solve.js
```

If you want to run it from the workspace root, use the full path:

```bash
node Day01-Task/solve.js
```
